    City of Chicago (CoC) has chosen Oracle Cloud Infrastructure to migrate their on-premises E-Business Suite workload to cloud and upgrade the E-Business Suite application version to 12.2.11 and Database version to 19c. This will allow them to replace their legacy hardware and move from a Capex to Opex model.

    Dependencies -City of Chicago migration/upgrade project includes Oracle E-Business Suite applications and current integrations in-place. -City of Chicago has decided to secure environments on OCI using Palo Alto firewalls. The migration and upgrade activities will be started initially without Firewalls in place and the same will be configured at a later stage of the project. -Application migration and upgrade activities will be started only after the on-premises to OCI connectivity is established over a IPSec VPN connection.
